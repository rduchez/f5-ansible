bigip_sys_global - Manage BIG-IP global settings
++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.3


.. contents::
   :local:
   :depth: 2


Synopsis
--------

* Manage BIG-IP global settings.


Requirements (on host that executes module)
-------------------------------------------

  * f5-sdk
  * f5-sdk >= 3.0.9

Examples
--------

 ::

    
    - name: Disable the setup utility
      bigip_sys_global:
        gui_setup: disabled
        password: secret
        server: lb.mydomain.com
        user: admin
        state: present
      delegate_to: localhost
